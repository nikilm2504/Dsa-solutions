## 📝 Solution Format

Each solution includes:

* Problem Name
* Platform
* Difficulty
* Approach
* Time Complexity
* Space Complexity
* Java Implementation

Example:

```java
/*
Platform    : LeetCode

Difficulty  : Easy

```
